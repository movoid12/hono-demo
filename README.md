### Required:

- Node.js 22
- latest pnpm

to start:

```
pnpm install
pnpm dev
```

## Folder Structure

```yml
src/
├── lib/
├── openapi/
├── middleware/
├── routes/
├── services/
├── utils/
├── app.ts
└── index.ts
```

Check out:

```py
// API-INDEX:
http://localhost:3000

// API Documentation:
http://localhost:3000/reference

// API Doc:
http://localhost:3000/doc
```
